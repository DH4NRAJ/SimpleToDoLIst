# SimpleToDoLIst
Basic ToDOList Using PERN Stack 



## Features

- ToDoList
- DarkMode
- Edit and Delete Notes

## Tech

- React JS
- Node JS
- Postgresql
- Express JS


## Installation

clone git repo 

```sh
cd client
npm i 
cd server
npm i
```

create postgres database using 

/server/todo.sql 


Use two terminals

```sh
nodemon index.js
npm start
```
